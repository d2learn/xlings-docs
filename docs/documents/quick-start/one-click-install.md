# 一键安装

> 在命令行窗口执行一键安装命令

### Linux/MacOS

::: code-group

```bash [curl]
curl -fsSL https://d2learn.org/xlings-install.sh | bash
```

```bash [wget]
wget https://d2learn.org/xlings-install.sh -O - | bash
```

:::

### Windows - PowerShell

```bash
irm https://d2learn.org/xlings-install.ps1.txt | iex
```
